openGL morden 방식으로 구성하기 위한 새로운 프로젝트 
- include/glad/glad.h 복사되어 있어야. 
- cmake 파일에서 glfw, glad라이브러리 설정 되어 있어야. 
    소스 인클루드 할때 CMakeLists내에 " src/*.c" <-- 공백 때문에 경로 인식 않되서 삽질. 

- github에 소스 반영해는 것 처음부터 다시 해봄.     
1. 저장소 초기화 : git init - .git 저장소가 생기면서  변화를 감시. 
2. 파일 담기 :    git add . ( . 폴더내의 모든 파일 을 다 담겠다는 말.  )
3. 기록 하기 :    git commit -m "Project start: CppOpenGLTest initial setup"
4. 원격저장소 새로 만들고 연결하기. 
    저장소 만들기: https://github.com/redfullmoon9-cmd/cppOpenGLTest.git
    로컬에서 원격 저장소로 연결: git remote add origin https://github.com/redfullmoon9-cmd/cppOpenGLTest.git
5. 브랜치 이름 변경 :    git branch -M main
6. 현재의 소스를 서버로 전송: git push -u origin main , 이제 부터 브랜치는 main이고 깃의 origin/main 이라고 설정. 
7. 올릴 필요 없는 파일들 build 결로 같은 경우.  ".gitignore" 파일(워킹 루트에 )만들어서 올려야.  
8. 올라가 파일 제거 하기. : git rm -r --cached build
9. 깃이그노어 파일 올리기: git add .gitignore 
10  커밋하기 : git commit -m "Docs:Add .gitignore file"
11. 서버 반영: git push
